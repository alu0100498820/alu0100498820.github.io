- Instalacion de Node.JS y Express (en linux)

Comenzaremos este tutorial con la instalacion de Node.js, para ello abrimos una terminal y escribimos lo siguiente:

  1. Primero instalaremos el PPA con el fin de obtener acceso a su contenido, para ello ejecutamos: $ curl -sL https: // deb.nodesource.com/setup | sudo bash -

  2. Despues de ejecutar el script de configuracion de nodesource, instalamos el paquete Node.js ejecutando: $ apt-get install nodejs

- Instalacion y Configuracion de Git (en linux)

Para instalar Git tenemos que volver a abrir una terminal y ejecutar el comando:

$ apt-get install git

A continuacion configuramos Git (nombre de usuario y dirección de correo), para ello ejecutamos:

1. $ git config --global user.name "(nombre de usuario)"
2. $ git config --global user.email (dirección de email)

Una vez configurado el nombre y la dirección toca configurar la herramienta de diferencias por defecto, usada para resolver conflictos de union (merge). Para ello ejecutamos

3. $ git config --global merge.tool vimdiff

A continuacion abrimos una cuenta en GitHub.

imagen3

Una vez abierta la cuenta tenemos que establecer conexion entre nuestro ordenador y el servidor de Git. Para ello tenemos que generar una clave .ssh, para ello ejecuatamos el comando:

$ ssh-keygen

la consola nos dira donde donde donde queremos que se guarde (lo dejare en la configuracion prederterminada).

Lo siguiente que nos pedira es una contraseña -passphrase- (la contraseña es opcional.)

Para leer la clave ssh ejecutaremos:

$ cat ~/.ssh/id_rsa.pub

luego copiamos la contraseña y lo pegaremos en SSH Keys en GitHub

### - Instalacion de PanDoc Para instalar PanDoc simplemente ejecutamos:

$ apt-get install pandoc

imagen6

Una vez instalado podemos convertir un fichero Markdown (extendido o estricto) a formatos como: + HTML + LaTeX + RTF + OpenDocument XML, etc.

Para ello tendremos que ejecutar:

$ pandoc -o archivo-de-salida.ext -i archivo-original.md
